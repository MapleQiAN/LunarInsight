## 🎨 设计系统

### 配色方案
- **主色调**：紫色渐变 (#667eea → #764ba2)
- **成功色**：绿色渐变 (#10b981 → #059669)
- **警告色**：黄色渐变 (#f59e0b → #d97706)
- **信息色**：蓝色渐变 (#3b82f6 → #2563eb)
- **文字颜色**：深灰 (#1e293b) / 中灰 (#64748b)
- **背景色**：浅灰渐变 (#f5f7fa → #e9ecef)

### 间距系统
- 小间距：8px、12px
- 中间距：16px、20px、24px
- 大间距：32px、40px、48px

### 圆角系统
- 小圆角：8px（标签、输入框）
- 中圆角：12px（按钮、小卡片）
- 大圆角：16px（主卡片）
- 圆形：50%（头像、图标背景）

### 阴影系统
- 轻阴影：`0 2px 8px rgba(0, 0, 0, 0.06)`
- 中阴影：`0 4px 16px rgba(0, 0, 0, 0.08)`
- 重阴影：`0 8px 24px rgba(0, 0, 0, 0.12)`
- 彩色阴影：`0 6px 20px rgba(102, 126, 234, 0.4)`

### 动画效果
- **缓动函数**：`cubic-bezier(0.4, 0, 0.2, 1)`
- **持续时间**：0.3s（快速）、0.5s（标准）
- **关键帧动画**：
  - `fadeIn`：淡入
  - `slideIn`：滑入
  - `bounceIn`：弹跳进入
  - `pulse`：脉冲
  - `float`：浮动
